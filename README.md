# 🛍️ Superstore Sales & Profit Analysis - Exploratory Data Analysis

## 📌 Project Overview

This project performs an exploratory data analysis (EDA) on the Superstore dataset to understand sales and profitability across different dimensions such as Region, Category, and Sub-Category.

The analysis focuses on identifying patterns that impact profit, including discount behavior, product performance, and regional differences. The goal is to uncover actionable insights that can help improve overall business profitability.

---

## 🎯 Objectives

Analyze total Sales and Profit across:

* Region
* Category
* Sub-Category

Identify:

* Top and bottom performing sub-categories based on profit
* Sub-categories generating losses
* Relationship between Discount and Profit
* Year-over-Year sales trends (2014–2017)

Evaluate:

* Profit margins across categories
* Regional performance differences

---

## 🛠️ Technologies Used

* Python
* pandas
* matplotlib
* seaborn

---

## 📊 Key Visualizations

* Bar Chart: Profit by Sub-Category (highlighting losses)
* Scatter Plot: Discount vs Profit
* Line Chart: Yearly Sales Trend (2014–2017)
* Bar Chart: Profit Margin (%) by Category
* Heatmap: Profit by Region and Category

---

## 📈 Key Insights

* Some sub-categories generate negative profit, indicating loss-making products that need attention.
* Higher discounts are strongly associated with lower profitability and potential losses.
* Certain categories have high sales but low profit margins, reducing overall efficiency.
* Profit performance varies across regions, showing opportunities for targeted improvements.

---

## 📂 Dataset

The dataset contains:

* 9994 rows and 21 columns
* No missing values
* Sales transaction data including Orders, Customers, Products, and Shipping details
